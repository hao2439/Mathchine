# Mathchine

## 📱 项目概述

**Mathchine** 是一款专属 Android 平台的数学练习应用，致力于帮助用户通过日常练习保持和提升数学计算与推导能力。从基础的加减乘除到进阶的高等数学，这里都有相应的练习和挑战。

## ✨ 核心特性

- **多领域支持**：涵盖加减乘除（四则运算）、线性代数、导数、概率论等多种数学分类。
- **专业公式展示**：采用规范的 LaTeX 格式进行题目的存储与展示，保证数学公式的严谨与排版美观。
- **离线数据包架构**：题目按分类以“数据包”的形式进行维护。客户端一次性下载某一类别的题目数据包后，无需持续联网即可随时随地刷题。
- **极简主义体验**：第一版 UI 遵循极简设计原则，去除所有花里胡哨的多余设计，专注最核心的“选包 - 刷题 - 反馈”流程。
- **高度可扩展**：无论是在开发环境还是使用阶段，都能够灵活、便捷地添加全新的数学种类或具体题目。
